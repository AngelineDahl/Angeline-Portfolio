# Tempe Crash Data Analysis (Lab 5)

## Overview
This project analyzes Tempe crash data to identify patterns in accident locations, time-of-day trends, age group risks, and harmful accident types. The goal is to support injury reduction strategies and improve public safety.

- dplyr  
- Data wrangling  
- Visualization tools (optional)

## Key Questions
- Which locations experience the most crashes?
- Are there specific times of day with higher crash frequency?
- Which age groups are most at risk?
- What harmful events contribute most to injuries?
