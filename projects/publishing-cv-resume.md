# Publishing My CV & Résumé with GitHub Pages

## Overview
This project involved creating online, shareable versions of my CV and résumé using GitHub Pages. The goal was to build a professional online presence with clean URLs and version-controlled documents.

## Tools Used
- GitHub Pages  
- Markdown  
- HTML  

## What I Did
- Created separate repositories for my CV and résumé  
- Converted documents into Markdown/HTML  
- Published each using GitHub Pages  
- Ensured clean, accessible URLs  

## Outcome
A polished, professional online presence with easily shareable links.
